# Maximus Enciclopédico Países

O skill permite buscar informações de países via API Rest Countries (https://restcountries.com/). Quando o usuário solicita dados de algum país, a skill deve consultar a API, extrair informações importantes como capital, população, idiomas, moedas, fusos e fronteiras, e apresentar esses dados de forma clara e adaptada ao tom informal preferido pelo usuário.
